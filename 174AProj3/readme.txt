Hospital Databases Class Project:

-Created and implemented a hospital database for a UCSB class. Designed schema and implemented it using MySQL
-Wrote a Java/JDBC module to import data into the database from universal table.
-Created 3 different user interfaces for the database, implemented using JavaFX.

To Run: 
Put the mysql/jdbc connector in your build path, execute the hospitaldb and healthmessagesexhange sql files, and press play!
